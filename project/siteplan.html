Project Site Plan

Site Name: Church Attendance Manager

•This name gives the idea of an application to manage attendance in Church.

Site Purpose:

•The application provides a way to add people, to visualize the directory, and to mark the attendance to Sunday meetings.

Scenarios:

•How can I add people to the directory? There will be a form page dedicated to adding people.

•How can I mark the attendance? The page attendance will serve that purpose. It will allow to mark the attendance for the last 10 weeks..

Color Schema:
#394f77 as the main color.
#e3e5da for text.
#f39c12 to underline the menu items.

Typography: "Montserrat", sans-serif
